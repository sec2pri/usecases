# Usecases

This project provides an overview of several cases where secondary identifiers are used, which could lead to information loss, hampering data integration with other resources, and hindering downstream data analysis methods. 

These cases are based on:

1. Publications
   1. Use of HGNC symbols in EuroPubMed Central
2. Databases
   1. Use of HGNC symbols in KEGG, Reactome, WikiPathways, Gene Ontology
3. Datasets
   1. transcriptomics
   2. metabolomics
